# Biography 

## Description of the Program 
The Biogragraphy program prints my name, hometown and my age on the console.
I have created this program using basic C++ Syntax (The rules of programming language-Example: #include <iostream>,Curly braces {} and simicolon ;)  , including variables,data types and output statements that i have learned during the Code Lab 1 lessons


## How to Run the program 

- Open VS Code and find the project folder named **Programming Skills Portfolio**    
- Open the terminal ( Ctrl + `` or view Terminal ).  
- Compile file by typing g++ Biograpgy.cpp  -0 Biograpgy    

### Execute Program
- On Windows:
.\Biograpgy.exe

- On macOS/Linux:
./Biograpgy
