# Is It Even

##  Description of the Program

**Is It Even** program allows the user to enter a number and checks if the number it is even or odd using a function **string checkEven(int num)**.
- Take an integer as input.
- Checks if the number can be divided by 2.
-Returns the a string saying whether the number is **even or odd**.

The actual progarm asks the user for a number then  calls the **checkEven()** function and displays the result on the Output.
